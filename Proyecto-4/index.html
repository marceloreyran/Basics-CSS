Ordenador

Definiciones

  • Ordenador : Dispositivo electrónico que permite almacenar información y procesarla para obtener resultados.Es un dispositivo electrónico que permite almacenar información y procesarla para obtener resultados.

  • Ordenador Compatible : Ordenador que puede ejecutar programas diseñados para otro ordenador, generalmente un IBM PC. Un ordenador que tiene la capacidad de almacenar y procesar información.

  • Ordenador Personal : Ordenador de sobre mesa que se utiliza en el hogar o en la oficina para realizar tareas Tipo de computadora portátil que se utiliza en el hogar o en la oficina para realizar tareas.

  • Ordenador Portatil : Ordenador que se puede llevar de un lugar a otro con facilidad, ya que es pequeño y ligero, y un portátil es una computadora que se puede llevar de un lugar a otro con facilidad.
